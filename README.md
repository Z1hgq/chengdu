# 2018暑期实习项目

## 成都榜样管理平台

- #### 整个平台分前台页面和后台管理页面，前台页面独立设计，后台页面采用了一个导航模版（这个模版真的垃圾，优化一点都不好，一缩放网页就乱了，改bug已修复）

- #### 主题采用bootstrap框架完成

- #### js框架为jQuery

- #### 后端语言采用php7

- #### 数据库采用mysql8

- #### 服务器采用Apache2

- #### 开发软件采用Atom、Chrome

- #### 六个人的队伍我一个人写代码五个人写文档=。=
